# Basic Android Firebase CRUD

### Features
- Add Artist
- Update Artist
- Delete Artist
- Add Track

### How to use
1. clone repo
```
$ git clone https://github.com/bailram/AndroidFirebaseCRUD.git
```
2. open android project folder with Android Studio
3. build project
